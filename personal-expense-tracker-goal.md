Project: Personal Expense Tracker
Well-defined goals:
Build a command-line expense tracker with these features:

Add new expense (amount, category, date, description)
View all expenses
View total spending
View spending by category
Delete an expense
Save/load data to a JSON file

Requirements (tests basics well):

Use lists + dictionaries
Functions for each feature
Loops and if/else logic
Input validation
JSON for saving data (json module)
Simple menu system
